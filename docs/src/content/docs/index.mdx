## Why VibeDepot?

<CardGrid stagger>
	<Card title="AI Providers Built In" icon="star">
		First-class support for Anthropic, OpenAI, and Gemini. Apps call AI through a unified Bridge API — users bring their own API keys.
	</Card>
	<Card title="Sandboxed & Secure" icon="approve-check-circle">
		Every app runs in an isolated Electron BrowserWindow with context isolation. Apps can only access what their permissions allow.
	</Card>
	<Card title="Permission System" icon="setting">
		Apps declare the permissions they need. Auto-granted storage, consent-required AI, database, network, clipboard, and notifications.
	</Card>
	<Card title="Public Registry" icon="open-book">
		A community-driven app catalog hosted on GitHub. Submit your app via PR, and it becomes available to every VibeDepot user.
	</Card>
	<Card title="Developer CLI" icon="laptop">
		Scaffold, preview, validate, and publish apps from the command line. Templates for vanilla HTML, React, chat bots, and more.
	</Card>
	<Card title="Open Source" icon="github">
		MIT licensed. Built with Electron, React, TypeScript, and Zustand. Contributions welcome.
	</Card>
</CardGrid>
