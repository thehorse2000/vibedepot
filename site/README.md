# VibeDepot site

Next.js app for the public **landing page** and **documentation**.

Lives in the VibeDepot monorepo at `site/` (same git history as the desktop app).

## Commands

```bash
pnpm install
pnpm dev      # http://localhost:3000
pnpm build
pnpm start
```

No app routes are defined yet beyond the root layout and page shell.
